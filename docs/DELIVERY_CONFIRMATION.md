## Customization Examples

### Change Primary Color
In `MeetingEnhanced.css`:
```css
:root {
  --primary-gradient: linear-gradient(135deg, #YOUR_COLOR_1 0%, #YOUR_COLOR_2 100%);
}
```

### Adjust Sidebar Width
In `MeetingEnhanced.jsx`, find:
```jsx
<div className="w-[420px]">  // Change this to desired width
```

### Change Button Size
In `MeetingEnhanced.jsx`, modify `PremiumButton` props:
```jsx
<PremiumButton size="lg" ... />  // or "md", "sm"
```

### Speed Up Animations
In `MeetingEnhanced.css`, change duration:
```css
animation: slideInUp 0.2s ease-out;  // Faster than 0.3s
```
