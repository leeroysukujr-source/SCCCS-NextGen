- **Primary**: Blue (#3B82F6) - Actions & active states

- **Success**: Green (#10B981) - Online, speaking
- **Alert**: Amber (#F59E0B) - Warnings
- **Error**: Red (#EF4444) - Recording, leave
- **Surfaces**: Slate gradients - Backgrounds

### Typography
- **Font**: Inter (modern, professional)
- **Sizes**: Carefully scaled hierarchy
- **Weights**: 400-800 for hierarchy
- **Spacing**: Generous and breathable

### Spacing
- **Base Unit**: 4px
- **Common**: 8px, 16px, 24px, 32px gaps
- **Button Padding**: 16px horizontal, 8px vertical

### Animations
- **Duration**: 0.3s standard transitions
- **Easing**: cubic-bezier(0.4, 0, 0.2, 1)
- **FPS**: Target 60fps on all devices
- **Effects**: Slide, fade, scale, pulse, glow
