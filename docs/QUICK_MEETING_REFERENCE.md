## Using New Features

### Switch Cameras
1. Click **More (⋯)** button
2. Select camera from dropdown
3. Camera switches instantly

### Approve Participants (Host Only)
1. Click **Participants (👥)** tab
2. Look for "Pending Approvals" section
3. Click **[✓]** to approve or **[✗]** to reject
4. Participant added/removed automatically

### Access Noise Suppression (Host Only)
**Option 1**: More menu
- Click **More (⋯)**
- Toggle **Noise Suppression** checkbox

**Option 2**: Settings modal
- Click **Settings (⚙️)**
- Look for "Host Controls" section
- Toggle **Noise Suppression**

### Use More Menu
1. Click **More (⋯)** button
2. Choose any option:
   - 📹 Switch Camera
   - ✓ Noise Suppression (admin)
   - 🔇 Mute All (admin)
   - 📐 Layout
   - 📊 Stats
   - ⚙️ Device Settings
   - 📞 End Meeting (admin)

### End Meeting (Host Only)
1. Click **More (⋯)**
2. Click **End Meeting** (red button)
3. All participants disconnected

## Status Indicators

### Video Grid
- **Blue Border + "Now Speaking"**: Current speaker
- **Amber Border + ✋**: Hand raised participant
- **Gray Border**: Inactive participant
- **Green Dot**: Currently speaking

### Participants Tab
- **Green dot**: Speaking now
- **✋ Badge**: Hand raised
- **Green 📹**: Camera on
- **Red 📹**: Camera off
- **Green 🎤**: Microphone on
- **Red 🎤**: Microphone off

### Header
- **HD**: Video quality indicator
- **Signal Bars**: Network connection
- **Host**: Meeting initiator badge
- **✋ Count**: Number of raised hands

## Troubleshooting

### Can't Switch Cameras
- Check if multiple cameras connected
- Refresh browser
- Check device permissions
- Try different camera port

### Approval Stuck Pending
- Host may not have approved yet
- Try rejoining
- Host can check Participants tab
- Contact meeting organizer

### More Menu Not Opening
- Click More button again
- Try refreshing page
- Clear browser cache
- Use incognito mode

### Can't See Host Controls
- You're not the meeting initiator
- Only hosts see admin features
- Contact meeting host for actions
- Join as host next time

### Noise Suppression Not Working
- Requires host to enable
- Check if microphone connected
- Restart browser
- Try different microphone
